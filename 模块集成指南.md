## 一、新增模块概览

### 1.1 模块清单（第一批）

| 模块名称 | 文件名 | 功能描述 | 优先级 |
|----------|--------|----------|--------|
| 图表智能推荐器 | chartRecommender.js | 基于数据特征和意图推荐最佳图表 | P0 |
| 错误收集器 | errorCollector.js | 自动捕获和记录系统错误 | P0 |
| 用户反馈管理器 | feedbackManager.js | 收集用户反馈和评价 | P0 |

### 1.2 模块清单（第二批）

| 模块名称 | 文件名 | 功能描述 | 优先级 |
|----------|--------|----------|--------|
| 数据预处理器 | dataPreprocessor.js | 单位转换、公式计算、数据格式化 | P1 |
| 自动修复器 | autoFixer.js | 定时自动修复已知问题 | P1 |

### 1.2 模块依赖关系

```
script.js (主程序)
├── chartRecommender.js (图表推荐)
│   └── 无依赖
├── errorCollector.js (错误收集)
│   └── 无依赖
├── feedbackManager.js (用户反馈)
│   └── errorCollector.js (可选，用于错误反馈关联)
├── dataPreprocessor.js (数据预处理)
│   └── 无依赖
└── autoFixer.js (自动修复)
    ├── errorCollector.js
    └── feedbackManager.js
```

### 2.1 图表智能推荐器 (chartRecommender.js)

#### 引入模块

```javascript
import chartRecommender from './js/chartRecommender.js';
```

#### 基本用法

```javascript
// 在生成图表前调用推荐
const dataInfo = {
    columns: ['省公司', '日期', '金额', '数量'],
    sampleData: [...],
    rowCount: 1000
};

const intentResult = {
    intent: 'QUERY_AGGREGATE',
    confidence: 0.85
};

const userInput = '统计各省公司的平均金额';

// 获取推荐结果
const recommendations = chartRecommender.recommend(dataInfo, intentResult, userInput);

console.log('推荐图表:', recommendations);
// 输出示例：
// [
//   { chartType: 'bar', reason: '适合分组对比', confidence: 0.8, source: 'intent_type' },
//   { chartType: 'pie', reason: '适合占比分析', confidence: 0.7, source: 'intent_type' }
// ]

// 使用推荐结果生成图表
if (recommendations.length > 0) {
    const bestChart = recommendations[0];
    console.log(`最佳推荐：${bestChart.chartType} (${bestChart.reason})`);
    
    // 使用推荐的 chartType 生成图表配置
    const chartConfig = {
        type: bestChart.chartType,
        // ...其他配置
    };
}
```

#### 在 script.js 中的集成点

```javascript
// 在 script.js 的 executeChartQuery 函数中
async function executeChartQuery(intent, userInput) {
    // ...现有代码...
    
    // 【新增】使用图表推荐器

        columns: headers,

    const recommendations = chartRecommender.recommend(dataInfo, intent, userInput);

        // 如果原配置没有指定图表类型，使用推荐的
        if (!chartConfigs[0].chartType || chartConfigs[0].chartType === 'bar') {
            chartConfigs[0].chartType = recommendedChartType;
        }
    }
    
    // ...继续执行图表生成...

```

### 2.2 错误收集器 (errorCollector.js)

#### 引入模块

```javascript
import errorCollector from './js/errorCollector.js';

// 挂载到全局（便于其他模块访问）
window.errorCollector = errorCollector;
```

#### 基本用法

```javascript
// 1. 手动记录错误
try {
    // 可能出错的代码
    const config = queryConfigGenerator.generateConfig(intent, userInput, headers);
    if (!config) {
        throw new Error('配置生成失败');
    }
} catch (error) {
    // 记录错误
    errorCollector.record(error, {
        userInput: userInput,
        intent: intent,
        module: 'queryConfigGenerator',
        action: 'generateConfig'
    });
    
    // 继续错误处理...
}

// 2. 自动捕获全局错误（已自动设置）
// window.addEventListener('error', ...) 已自动监听
// window.addEventListener('unhandledrejection', ...) 已自动监听

// 3. 获取未提交的错误
const unsubmittedErrors = errorCollector.getUnsubmittedErrors();
console.log('未提交错误:', unsubmittedErrors);

// 4. 导出错误报告
const report = errorCollector.exportReport();
console.log('错误报告:', report);

// 5. 下载错误报告
errorCollector.downloadReport();
```

#### 在 script.js 中的集成点

```javascript
// 在 script.js 的 handleUserInput 函数中
async function handleUserInput(userInput) {
    try {
        // 记录开始处理
        console.log('[处理] 开始处理用户输入:', userInput);
        
        // 意图识别
        const intent = await intentRecognizer.recognize(userInput, {
            columns: headers,
            rowCount: data.length
        });
        
        // 执行查询或图表生成
        if (intent.intent.startsWith('CHART_')) {
            await executeChartQuery(intent, userInput);
        } else {
            await executeQuery(intent, userInput);
        }
        
    } catch (error) {
        // 【新增】记录错误
        errorCollector.record(error, {
            userInput: userInput,
            module: 'handleUserInput',
            action: 'process_request'
        });
        
        // 显示错误提示
        showError(`处理失败：${error.message}`);
    }
}

// 在图表生成函数中
async function executeChartQuery(intent, userInput) {
    try {
        // ...现有代码...
    } catch (error) {
        // 【新增】记录错误
        errorCollector.record(error, {
            userInput: userInput,
            intent: intent,
            module: 'chartGenerator',
            action: 'executeChartQuery',
            dataInfo: { columns: headers, rowCount: data.length }
        });
        throw error;
    }
}
```

#### API 文档

```javascript
// 记录错误
errorCollector.record(error, context)
// 参数：
// - error: Error 对象或包含 message 的对象
// - context: {userInput, intent, module, action, dataInfo, config}
// 返回：errorId (String)

// 获取未提交的错误
errorCollector.getUnsubmittedErrors()
// 返回：Array<ErrorRecord>

// 获取所有错误（带过滤）
errorCollector.getAllErrors({type, module, isSubmitted, limit})
// 返回：Array<ErrorRecord>

// 标记为已提交
errorCollector.markAsSubmitted([errorId1, errorId2, ...])

// 标记为已修复
errorCollector.markAsFixed([errorId1, errorId2, ...], fixMethod)

// 导出错误报告
errorCollector.exportReport()
// 返回：{exportTime, totalErrors, statistics, errors, systemInfo}

// 下载错误报告
errorCollector.downloadReport()

// 清除错误
errorCollector.clear([errorId1, errorId2, ...]) // 不传则清除所有

// 获取统计信息
errorCollector.generateStatistics()
// 返回：{totalErrors, byType, byModule, byDate, recentTrends}

// 获取数量统计
errorCollector.getCounts()
// 返回：{total, submitted, unsubmitted, fixed, unfixed}
```


```html
function initFeedbackUI() {
    const feedbackSection = document.querySelector('.feedback-section');
    const btnFeedbackIncorrect = document.getElementById('btnFeedbackIncorrect');
    const stars = document.querySelectorAll('.star');
    const btnSubmitSuggestion = document.getElementById('btnSubmitSuggestion');
    const btnErrorReport = document.getElementById('btnErrorReport');
    const btnFeedbackReport = document.getElementById('btnFeedbackReport');
    
    // 显示反馈区域

        feedbackSection.style.display = 'block';
    }
    
    // 识别有误按钮

        btnFeedbackIncorrect.addEventListener('click', () => {
            const expectedIntent = prompt('请描述您期望的操作（例如：筛选广东数据）：');
            if (expectedIntent && window.currentQuery) {
                window.feedbackManager.quickFeedbackIncorrect(
                    window.currentQuery.userInput,
                    window.currentQuery.intent,
                    expectedIntent
                );
                showToast('感谢您的反馈！我们将持续改进');
            } else if (!window.currentQuery) {
                showToast('请先执行一次查询');
            }
        });

    // 评分星星

        star.addEventListener('click', () => {

            });

        });
    });
    
    // 提交建议
    if (btnSubmitSuggestion) {
        btnSubmitSuggestion.addEventListener('click', () => {
            const text = document.getElementById('suggestionText').value.trim();
            if (text) {
                window.feedbackManager.submitSuggestion(text, 'general');

                showToast('感谢您的建议！');
            } else {
                showToast('请输入建议内容');

    // 错误报告按钮
    if (btnErrorReport) {
        btnErrorReport.addEventListener('click', show_errorReport);
    }
    
    // 反馈统计按钮

// 显示错误报告
function show_errorReport() {
    const modal = document.getElementById('errorReportModal');
    const statsContainer = modal.querySelector('.error-stats');
    const listContainer = modal.querySelector('.error-list');
    
    // 获取统计信息
    const stats = window.errorCollector.generateStatistics();
    
    // 显示统计
    statsContainer.innerHTML = `
        <div class="stats-grid">
            <div class="stat-item">
                <div class="stat-value">${stats.totalErrors}</div>
                <div class="stat-label">总错误数</div>
            </div>
            <div class="stat-item">
                <div class="stat-value">${stats.byType['COLUMN_ERROR'] || 0}</div>
                <div class="stat-label">列错误</div>
            </div>
            <div class="stat-item">
                <div class="stat-value">${stats.byType['PARAM_ERROR'] || 0}</div>
                <div class="stat-label">参数错误</div>
            </div>
            <div class="stat-item">
                <div class="stat-value">${Object.keys(stats.byModule).length}</div>
                <div class="stat-label">涉及模块</div>
            </div>
        </div>
    `;
    
    // 显示错误列表
    const errors = window.errorCollector.getAllErrors({ limit: 50 });
    listContainer.innerHTML = `
        <table class="table">
            <thead>
                <tr>
                    <th>时间</th>
                    <th>类型</th>
                    <th>模块</th>
                    <th>错误信息</th>
                    <th>状态</th>
                </tr>
            </thead>
            <tbody>
                ${errors.map(error => `
                    <tr>
                        <td>${error.timestampFormatted}</td>
                        <td>${error.type}</td>
                        <td>${error.context.module}</td>
                        <td>${error.error.message}</td>

                        </td>
                    </tr>
                `).join('')}
            </tbody>
        </table>
    `;
    
    modal.style.display = 'block';
}

    const ids = errors.map(e => e.id);

});

// 辅助函数：显示提示
function showToast(message) {
    // 实现一个简单的 toast 提示
    const toast = document.createElement('div');
    toast.className = 'toast';
    toast.textContent = message;
    toast.style.cssText = `
        position: fixed;
        bottom: 20px;
        left: 50%;
        transform: translateX(-50%);
        background: rgba(0,0,0,0.8);
        color: white;
        padding: 10px 20px;
        border-radius: 5px;
```

```javascript
// 1. 启动定时自动修复（每 60 分钟执行一次）
autoFixer.scheduleAutoFix(60);

// 2. 手动触发修复指定错误
const errorIds = ['ERR_1234567890_abc', 'ERR_1234567891_def'];
const results = await autoFixer.triggerFix(errorIds);

//   totalFixes: 50,
//   successfulFixes: 35,
//   failedFixes: 15,
//   successRate: '70.00%',
//   byRule: {...},
//   registeredRules: 6,

// }

// 5. 查看修复历史
const history = autoFixer.getFixHistory(20);

// 6. 停止定时修复
autoFixer.stopAutoFix();

// 7. 注册自定义修复规则
autoFixer.registerRule({
    id: 'custom_rule_1',
    name: '自定义修复规则',
    description: '针对特定场景的修复',
    condition: (error) => {
        return error.context.module === 'custom_module';
    },
    fixAction: async (error, context) => {
        // 自定义修复逻辑
        return {
            success: true,
            action: 'custom_fix',
            details: '使用自定义规则修复'
        };
    },
    successRate: 0.8,
    priority: 1
});
```

#### 在 script.js 中的集成点

```javascript
// 在应用启动时初始化自动修复
document.addEventListener('DOMContentLoaded', () => {
    // ...其他初始化代码...
    
    // 【新增】启动定时自动修复

        // 每 60 分钟自动修复一次

    }

    const btn = document.getElementById('btnAutoFix');
    btn.disabled = true;
    btn.textContent = '修复中...';

        const successCount = results.filter(r => r.success).length;
```

#### API 文档

```javascript

// 重置统计信息
autoFixer.resetStatistics()
```

#### 修复规则说明

系统默认注册了 6 条修复规则：

1. **semantic_mapping_missing** - 语义映射缺失自动补充
   - 触发条件：列错误且消息包含"未能匹配到列名"
   - 修复动作：从用户输入中提取术语，映射到最相似的列名
   - 成功率：70%

2. **low_confidence_intent** - 低置信度意图识别优化
   - 触发条件：意图识别置信度 < 0.5
   - 修复动作：记录低置信度案例，用于后续模型优化
   - 成功率：90%

3. **param_extraction_failed** - 参数提取失败修复
   - 触发条件：参数提取失败
   - 修复动作：使用大模型 API 重新提取参数
   - 成功率：60%

4. **chart_type_mismatch** - 图表类型不匹配修复
   - 触发条件：图表类型不匹配数据
   - 修复动作：使用图表推荐器重新推荐
   - 成功率：80%

5. **data_format_error** - 数据格式错误修复
   - 触发条件：数据格式错误
   - 修复动作：使用数据预处理器修正
   - 成功率：50%

6. **user_feedback_driven** - 用户反馈驱动的修复
   - 触发条件：存在相关的用户反馈
   - 修复动作：根据用户期望的意图进行调整
   - 成功率：85%

## 五、常见问题

### Q1: 模块加载失败怎么办？

**A**: 确保使用 ES6 模块语法，并在 HTML 中使用`type="module"`:

```html
<script type="module">
    import chartRecommender from './js/chartRecommender.js';
</script>
```

### Q2: localStorage 存储满了怎么办？

**A**: 模块已自动处理，当存储超过上限时会删除最旧的记录。也可以手动清除:

```javascript
// 清除所有错误
window.errorCollector.clear();

// 清除所有反馈
window.feedbackManager.clear();
```

### Q3: 如何禁用错误收集或反馈收集？

**A**: 可以通过配置开关控制:

```javascript
// 在 config.js 中添加配置
window.config = {
    enableErrorCollection: true,  // 禁用错误收集
    enableFeedbackCollection: true // 禁用反馈收集
};
```
