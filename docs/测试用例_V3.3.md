## 1. 测试范围

### 1.1 功能测试范围
- 意图识别（15种意图类型）
- 数据查询（查找、统计、筛选、排序）
- 数据透视表
- 数据清洗
- 图表生成（8种图表类型）
- 数据导出
- 大数据处理

### 1.2 测试优先级
- P0：核心功能，必须通过
- P1：重要功能，应该通过
- P2：辅助功能，建议通过
