# 智能数据分析助手 V4.0 发布说明

## 发布日期
2026年3月2日

## 版本概述
V4.0是一次重大架构升级，引入了**分层Agent架构**，实现了功能职责的清晰分离，大幅提升了系统的可扩展性和智能化水平。

---

## 🚀 核心新特性

### 1. 分层Agent架构
引入四种专门的Agent，各司其职：

| Agent | 职责 | 核心能力 |
|-------|------|---------|
| **分析Agent** | 数据分析 | 查询、聚合、筛选、排序、分组、计算 |
| **图表Agent** | 可视化 | 图表推荐、配置生成、自定义、类型转换 |
| **解释Agent** | 结果解读 | 总结、洞察生成、建议提供 |
| **规划Agent** | 执行规划 | 计划生成、任务分解、可行性验证 |

### 2. 数据画像模块 (DataProfiler)
自动构建数据画像，让系统"理解"数据：

- **列类型推断**：自动识别数值型、文本型、日期型、分类型
- **数据质量评估**：完整性、唯一性、异常值检测
- **数据分布分析**：最大值、最小值、平均值、中位数、标准差
- **Schema推断**：自动识别维度列和指标列
- **相关性分析**：检测数值列之间的相关性

### 3. 执行计划展示
复杂查询前展示执行计划，提升透明度：

- 可视化展示执行步骤
- 显示预估执行时间
- 支持用户确认/取消
- 实时进度更新

### 4. 意图实体抽取增强
从用户输入中提取关键实体：

- **图表类型**：折线图、柱状图、饼图等
- **维度/指标**：自动识别分组列和数值列
- **时间范围**：今天、本周、近7天等
- **聚合方式**：求和、平均、计数等
- **排序方式**：升序、降序、Top N
- **筛选条件**：大于、小于、包含等

## 📝 使用示例

### 数据画像
```javascript
// 加载数据后自动生成画像
const profile = window.dataProfiler.profile(data, headers);
console.log(profile.summary);
// 输出：数据集共1000行×5列，数值型字段2个，分类字段1个，数据完整度98.5%，质量评级A。

// 获取LLM友好的描述
const desc = window.dataProfiler.getLLMDescription();
```

### Agent调度
```javascript
// 执行完整分析流程
const result = await window.agentRouter.executeFullPipeline(config, data, headers, dataProfile);

// 结果包含：
// - plan: 执行计划
// - analysis: 分析结果
// - chart: 图表配置
// - explanation: 解释说明
```

### 实体抽取
```javascript
// 从用户输入中提取实体
const entities = window.entityExtractor.extract('画一个近7天销售额的趋势图', columns);
// 返回：
// {
//   chartType: { type: 'line', confidence: 0.7 },
//   timeRange: { range: 'recent7days', keyword: '近7天' },
//   measures: [{ column: '销售额', confidence: 0.9 }]
// }
```
