### 2.1 定期分析功能

#### 现状
- **状态**: 未实现
- **完成度**: 0%

#### 需求描述
用户希望能够设置定期自动分析任务，系统按指定时间间隔自动生成分析报告并推送。

#### 使用场景
1. **日报**: 每天早上8点自动生成昨日销售数据分析报告
2. **周报**: 每周一早上生成上周业绩汇总报告
3. **月报**: 每月1号生成上月财务报表
4. **实时监控**: 每15分钟检查一次关键指标异常

#### 向量化流程
```python
class ExcelVectorizer:
    def __init__(self):
        self.embedding_model = SentenceTransformer('paraphrase-multilingual-MiniLM-L12-v2')
        self.vector_store = ChromaDB()
    
    def vectorize(self, df, table_name):
        """将DataFrame向量化存储"""
        
        # 1. 数据清洗和预处理
        clean_df = self._preprocess(df)
        
        # 2. 生成行向量（每行数据生成一个向量）
        for idx, row in clean_df.iterrows():
            # 将行数据转换为文本描述
            row_text = self._row_to_text(row)
            
            # 生成向量
            embedding = self.embedding_model.encode(row_text)
            
            # 存储到向量库
            self.vector_store.add(
                id=f"{table_name}_{idx}",
                embedding=embedding,
                metadata={
                    'table': table_name,
                    'row_idx': idx,
                    'data': row.to_dict()
                }
            )
        
        # 3. 生成列向量（每列生成语义向量）
        for col in clean_df.columns:
            col_text = f"列名: {col}, 类型: {clean_df[col].dtype}"
            embedding = self.embedding_model.encode(col_text)
            
            self.vector_store.add(
                id=f"{table_name}_col_{col}",
                embedding=embedding,
                metadata={
                    'table': table_name,
                    'column': col,
                    'type': 'column_meta'
                }
            )
    
    def query(self, query_text, filters=None, top_k=10):
        """语义查询"""
        
        # 1. 查询向量化
        query_embedding = self.embedding_model.encode(query_text)
        
        # 2. 语义扩展（理解地域、时间等）
        expanded_query = self._semantic_expansion(query_text)
        
        # 3. 向量检索
        results = self.vector_store.similarity_search(
            query_embedding,
            filter=filters,
            k=top_k
        )
        
        # 4. 结果后处理
        return self._post_process(results)
```
