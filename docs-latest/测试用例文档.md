## 二、意图识别算法改进

### 2.1 改进前的问题

```
用户输入: "华东的销售额是多少"
旧算法结果:
  - rField = "地区" (错误)
  - vField = "数量" (错误，应该是"销售额")
  - 输出: "华东地区数量: 300" (错误)
```

### 2.2 改进后的算法

```javascript
// 数值字段关键词 - 按优先级排序（精确匹配优先）
const valueKeywords = [
    '销售额', '销售总额', '销售金额',  // 精确匹配放前面
    '销售', '利润', '数量', '金额'      // 模糊匹配放后面
];

// 算法流程:
1. 检测用户输入中的关键词: ["华东", "销售额"]
2. 精确匹配: "销售额" -> 匹配"销售额"列
3. 避免"销售"同时匹配"销售额"和"数量"
4. 区域匹配: "华东" -> 匹配"省公司"列中的"华东"
```

### 2.3 改进后结果

```
用户输入: "华东的销售额是多少"
新算法结果:
  - rField = "省公司" (正确)
  - vField = "销售额" (正确)
  - 输出: "华东地区销售额: 7,400,000" (正确)
```
